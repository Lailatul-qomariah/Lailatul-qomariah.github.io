Oca' Keyasan

Oca' se ghi' ta' tegghes sabhenderra, kantos ghi' parlo esare artena/tegghessa sa samastena. Oca' keyas mebis kalaban oca' dhin-andhin. Oca' keyas reya eoca'aghi/eandhiaghi se tojjhuenna da' pola tengkana manossa.

A

Addhu semmo = Ngoca' kalaban asemmonan
Aotang rassa = Aghabay sake' atena oreng laen
Aeng odi' = Aeng aghili tada' nembhara' nemorra
Ate malam = cappet/lekkas sabbhar
Abalang lere = Alerek
Acolo' songay = Mellak
Ate kadungdung = Jhuba' babatekka
Aghantong ghighi = Kasusu se ngakana
Atena abulu = Sepatta burik/dhengghi/dhariya
Ajhuwal sengnga' = Nodhuaghi kasombonganna
Asalen tembhang = Asalen maksod
Ajhuwal kopeng = nyare kabher/berta
Amata duwa' = Abhanto moso
Ajhuwal caca = Andhun nyacae
Apangeran da' bato = Mojha bato
Amata pesse = Samane menta pessa
Ajhuwal omo' = Bennya' caca sopaja earane penter
Ajhuwal pello = Alako ngala' ongkosan
Ajhuwal ropa = Nodhuaghi ropana aba'na
Acolo' bine' = Soka' madapa' caca
Abhantal ombak = Alako se bhabhaja
Amaen pesse = Amaen kartu kalaban etaroe pesse
