Parebhasan

A

1. Aba' sampayan
Artena : Ekoca'aghi da' ka oreng apangangghuy apa'a bhei patot.

2. Aba' kamonduran
Artnena : Oreng se ta' noro'aghi da' sakabhianna kabada'an edimmaa bhai.

3. Acaca duwa'
Artena : Ta' pagghun cacana, ta' kenning eparcaje oreng

4. Adiguna
Artena : Ngandhelaghi kapenterranna.

B

1. Bibir attas bibir baba
Artena : Ce' ghempangnga le' male' caca

2. Bhellu abunto' are'
Artnena : Da'-ada'ne becce' san kabudina macalaka'a

3. Bibirra nola' atena mellak
Artena : Etare ta' enda' tape sanyatana enda'/terro

4. Bellas ta' asel
Artena : Ekaneserre oreng tape ta' ngontongngaghi
