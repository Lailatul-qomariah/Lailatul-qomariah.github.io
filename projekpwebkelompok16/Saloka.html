Saloka

A

Asel ta' adhina asal = Sanajjan soghi tape andhap asor

Adante' ghettana bato = Adante' bharang pa-apa se mohal ekaolle

Akopa' tanang salaja = Parembhaghan duwa' pehak se coma esaroju'i settong peha' mohal kalakon

Adante' nyellemma ghebhus = Adante' bharang pa-apa se bellun tanto ekaolle

B

Bada pakon, bada pakan = Ro-nyoro oreng kodhu eberri' pakan/opa

Bada cator mongkor = Tak ghellem ngedhingngaghi caca se becce'

C

Cangker ketendien pereng = Kala kobesa da' setowaan

Cetak ghundul e keppette = Aba'la nyaman ghi' etambei nyaman pole

Dangdang abulu merrak = Oreng asor apanrtengka newo oreng aghung

D

Dangdang abulu merrak = Oreng asor apanrtengka newo oreng aghung
